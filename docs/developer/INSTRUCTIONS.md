# Инструкции Разработчика (этап 5)

Исполняемые команды Cursor лежат в репозитории продукта:

| Команда | Назначение | Источник в эксперименте |
| --- | --- | --- |
| `/task <номер>` | Загрузка Issue, уточнения, реализация, QA | [`task.md`](task.md) |
| `/adr` | ADR по уже выполненной задаче (для Code Review) | [`adr.md`](adr.md) |

Типичный порядок: `/task 32` → (после зелёного QA) `/adr` → по указанию Пользователя MR, Code Review и сдача PM (разделы «Далее» в `task.md`).

Файлы в `.cursor/commands/` у Разработчика должны совпадать с `task.md` / `adr.md` здесь. При правке инструкции обновляй оба места.
